- What is HTTP?

Hypertext Transfer Protocol; it's how browsers and servers communicate. - What is a URL?

- What is DNS?

Document naming system; it takes the name of the website and translates it into an IP address

- What is a query string?

At the end of a url, the query string provides some extra information that can be used by the server to modify the page

- What are two HTTP verbs and how are they different?

GET and POST; get does not change the website at all, and post will make changes/create an item for example

- What is an HTTP request?

A request made by the user not the server to access something on the server

- What is an HTTP response?

An action from the server back to the user in response to the requested information.

- What is an HTTP header? Give a couple examples of request and response headers you have seen.

Part of a request that will include the hostname, date, language, cookies, etc.

- What are the processes that happen when you type “http://somesite.com/some/page.html” into a browser?

The host name will first be translated into an IP address for the user's device to access, then using HTTP protocol, a request will be made to access the header and page information which will be delivered by the server back to the user's browser.
